## Writing a new ADR

1. Copy the format of ADR-0001.
2. Number sequentially. Do not renumber after merge.
3. Status starts as `Proposed`. Move to `Accepted` on merge.
4. Link from [SPECIFICATION.md](../SPECIFICATION.md) under "Design
   decisions" if the decision is repo-wide.
